<h1>🎮 Tic-Tac-Toe Game-</h1>
Welcome to my Tic-Tac-Toe Game — a timeless, two-player browser-based game built with HTML, CSS, and JavaScript. Simple, fun, and responsive — this project showcases clean UI, interactive game logic, and winner detection, perfect for beginners and casual players!

<h1>✨ Features-</h1>
✅ Two-Player Mode — Play with a friend, taking turns on the same screen.<br>
✅ Responsive Layout — Smooth experience on desktop, tablet, and mobile.<br>
✅ Winner Detection — Instantly highlights the winning combination.<br>
✅ Draw Condition Handling — Notifies players if the match is a draw.<br>
✅ Restart Option — Easily reset the board and start a new game.<br>
✅ Minimal & Clean Design — Focused, distraction-free game board.<br>

<h1>🎨 Technologies Used-</h1>
HTML5 — Structure of the game.<br>
CSS3 — Styling the board, buttons, and layout.<br>
JavaScript (Vanilla) — Game logic, player turns, and result detection.<br>

<h1>📂 Project Structure-</h1>
tic-tac-toe-game/<br>
│<br>
├── index.html        # Game interface<br>
├── style.css         # Styling for the board, buttons, and layout<br>
├── script.js         # Core game logic and interactivity<br>
<h1>🚀 Getting Started-</h1>
Follow these simple steps to run the project locally:Clone the repository.<br>
git clone https://github.com/POWER-WORLD/tic-tac-toe-game.git<br>
Navigate to the project folder:<br>
cd tic-tac-toe-game<br>
Open index.html in your web browser<br>
